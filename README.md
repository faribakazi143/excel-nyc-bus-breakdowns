# NYC Bus Breakdowns & Delays Analysis

**Project Overview**  
Analyzed a large NYC school bus dataset to identify patterns in breakdowns and delays and provide insights for improving the transportation system.

**Tools & Features Used**
- Power Query for data cleaning
- PivotTables and Pivot Charts
- Interactive Slicers
- Summary statistics and trend analysis

**Key Insights**
- Heavy Traffic is the most common cause of delays
- Delay times vary significantly by bus company and borough
- Mondays have the highest number of incidents
